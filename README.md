Replication & Critique: ML for Credit Risk in the Reactiva Peru ProgramThis repository contains a full replication and methodological critique of the study "Machine Learning for Credit Risk in the Reactiva Peru Program: A Comparison of the Lasso and Ridge Regression Models" by Geraldo-Campos et al. (2022) .📌 Project HighlightsCritical Discovery: Diagnosed a target leakage flaw in the original experimental design where the credit risk target was mechanically derived from a primary regressor.Statistical Rigor: Validated the flaw using Kruskal-Wallis tests ($p < 10^{-300}$) and Random Forest models ($R^{2} = 1.0$).Large-Scale Data: Processed a dataset of 500,000+ SME loan records from the Peruvian government's COVID-19 relief program.🛠️ Technical ImplementationCore ReplicationModels: Re-implemented OLS, Lasso, and Ridge regressions in Python.Optimization: Automated hyperparameter tuning using LassoCV and RidgeCV.Pipeline: Built a robust preprocessing workflow including ordinal encoding for categorical variables (economic sector, lending entity) and MinMaxScaler normalization.Methodological ExtensionThe replication revealed that linear models achieved a weak $R^{2} \approx 0.08$, with performance largely insensitive to regularization strengths ($\alpha$). I extended the study to investigate why:Leakage Diagnosis: Proved the "Risk Level" was determined solely by "Covered Amount" thresholds.Non-Linear Validation: Deployed Random Forests to test if the rule could be recovered. The model achieved a perfect $R^{2}$ and RMSE $\approx 0$, confirming the deterministic relationship.Feature Importance: Visualized that "Covered Amount" provided 100% of the predictive power, rendering other regressors obsolete.📊 Key FindingsModelRMSER2NoteLasso (Optimal)0.35660.0823Replicated Original ResultRidge (Optimal)0.35660.0823Replicated Original ResultRandom Forest0.000071.0000Confirmed Target Leakage📁 Repository StructurePeru_Credit_Risk_Replication.ipynb: Main analysis, statistical tests, and modeling.Report.pdf: Comprehensive academic paper detailing the replication and findings.Reactiva_Peru.csv: Cleaned dataset of 501,298 SME loan records.🚀 How to RunClone the repo: git clone https://github.com/CatalinMoldova/Replication-and-Extension-of-Linear-Regressor-of-the-Reactiva-program-in-Peru.gitInstall dependencies: pip install scikit-learn pandas numpy matplotlib seaborn scipyExecute Peru_Credit_Risk_Replication.ipynb in a Jupyter environment.🎓 Academic ContextThis project was completed in December 2025 as part of the Machine Learning and Computational Statistics course at New York University Abu Dhabi.
